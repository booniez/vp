## 继承的特点

- 只支持 ***单继承***，不支持多继承，但是支持 ***多层继承***

::: tip
为什么 Java 不支持多继承？
因为如果同时继承多个父类，父类中都拥有相同都方法，子类该继承哪个方法呢？
![多继承都危害](/Java/WX20240416-170754@2x.png)
所以，Java 不支持多继承
:::

## 成员变量都访问特点

- 在子类方法中访问一个变量
    * 在子类局部范围查找
    * 在子类成员范围查找
    * 在父类成员范围查找
    
:::tip
在子父类中如果出现 **重名** 的成员变量，遵循 ***就近原则***，优先使用子类的成员变量，如果一定要使用父类的，则使用 ***super*** 进行区分
:::
